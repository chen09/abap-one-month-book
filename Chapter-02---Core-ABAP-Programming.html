本章进入 ABAP 程序员每天都会使用的核心能力：ABAP SQL、内部表、OO ABAP、Modern ABAP、基础测试与代码质量。对 Java/Python 背景的读者来说，本章最重要的转换不是“学一套新语法”，而是把数据访问、内存表处理、对象封装、返回码、异常、事务边界和 SAP 项目交付习惯放到同一张图里理解。

本章学习目标

  • 能解释 ABAP SQL 如何通过 ABAP database interface 访问 DDIC / CDS 数据模型。
  • 能读懂一段典型 SELECT，说明数据源、host variable、结果接收、排序和权限边界。
  • 能区分 internal table 的基础读写、循环、排序、查找和性能风险。
  • 能理解 OO ABAP 的 class、method、interface、exception 和 ABAP Unit 基础。
  • 能识别 Modern ABAP 表达式的可读性收益与滥用风险。
  • 能在日本 SAP 项目 review 中说明 検索条件、ソート順、権限チェック、トランザクション境界 等关键点。
